# GSAP

<b>gsap</b>이란? </br>

( GreenSock Animation Platform )

> greensock 회사에서 만든 애니메이션 플랫폼으로, JavaScript가 만질 수 있는 모든 것 (CSS속성, SVG, React, canvas, normal, object 등) 에 애니메이션을 적용하고 수많은 브라우저들의 크로스브라우징을 지원하며 jQury보다 최대 20배 빠른 성능을 제공합니다.
